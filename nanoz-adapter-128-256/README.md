# nanoz-adapter-128-256
Multiplexed adapter to interface eib-128 and eib-256 with the popular
[nanoZ](http://www.white-matter.com/nanoz/) electrode impedance tester and
plating device.

![nanoz-128](./img/nanoz-adapter-128-256.png)
